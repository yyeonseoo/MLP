# New ML Project

머신러닝 프로젝트 템플릿입니다.

## 구조

```
new_ml_project/
├── data/           # 데이터
│   ├── raw/        # 원본 데이터
│   └── processed/  # 전처리된 데이터
├── src/            # 소스 코드
├── scripts/        # 실행 스크립트
├── notebooks/      # Jupyter 노트북
├── outputs/        # 산출물
├── requirements.txt
└── README.md
```

## 설정

```bash
pip install -r requirements.txt
```
